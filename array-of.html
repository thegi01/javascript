Array.of(element0[, element1[, ...[, elementN]]])

The The Array.of() method creates a new Array instance from a variable number of arguments, regardless of number or type of the arguments.

The difference between Array.of() and the Array constructor is in the handling of interger arguments: Array.of(7) creates an array with a single element, 7, whereas Array(7) creates an empth with a length property of 7 (Note: this implies an array of 7 empth slots, not slots with actual undefined values).

Parameters

  • elementN: Elements of which to create the array.

반환값

A new Array instance.
