string.prototype.slice ( start, end )

The slice method takes two arguments, start and end,
and returns a substring of the result of converting this object to a String,
starting from index start and running to, but not including, index end (or through the end of the String if end is undefined).
If start is negative, it is treated as sourceLength+start where sourceLength is the length of the String.
If end is negative, it is treated as sourceLength+end where sourceLength is the length of the String.
The reusult is a String value, not a String object.
