Object.keys

Syntax

Object.keys(obj)

Parameters

obj
The object of which the enumerable own properties are to be returned.
Retrun value
An array of strings that represent all the enumerable properties of the given object.

설명

Object.keys() returns an array whose elements are strings corresponding to the enumerable properties found directly upon object.

The ordering of the properties is the same as that given by looping over the properties of the object manually.

Object.keys()는 object에서 직접 검색된 열거형 속성들에 대응하는 문자열들을 요소로 갖는 배열을 반환합니다.

반환된 요소는 객체의 요소 전체를 수동으로 반복(loop)하며 주워지는 순서와 동일합니다.
