window.history

The History interface allows maipulation of the browser session historty, this is the pages visited in the tab or frame that the current page is loaded in.

Propertis

History.length
Returns an Integer representing the number of elements in the session history, including the currently loaded pages. For example, fora page loaded in a new tab this property retruns a.
History.scrollRestoration
Allows web applications to explicitly set default scroll restoration behavior on history navigation. This property can be either auto or manual. This property can be either auto or manual.
History.state
Returns an any value representing the state at the top of the history statck. This is a way to look at the state without having to wait for a popstate event.
